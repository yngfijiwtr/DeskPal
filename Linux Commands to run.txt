*** Raspberry Pi running the latest Raspbian OS (Debian based Linux) ***
*** Set it up for SSH if you would prefer that, here's a tutorial: https://www.onlogic.com/blog/how-to-ssh-into-raspberry-pi/ ***

***update and install Python***
sudo apt-get update
sudo apt-get -y upgrade
sudo apt-get install python3-pip

sudo apt install --upgrade python3-setuptools

cd ~
pip3 install --upgrade adafruit-python-shell --break-system-packages

wget https://raw.githubusercontent.com/adafruit/Raspberry-Pi-Installer-Scripts/master/raspi-blinka.py
sudo -E env PATH=$PATH python3 raspi-blinka.py

pip3 install adafruit-circuitpython-charlcd --break-system-packages

pip3 install yfinance --break-system-packages
pip3 install cryptocompare --break-system-packages

sudo apt-get install libopenblas-dev

cd
mkdir Logs

sudo nano /etc/systemd/system/deskpal.service

Add the following text to the file:
"
[Unit]
Description=Deskpal
After=network.target

[Service]
User=pi

# This is the full path to your streamer directory
WorkingDirectory=/home/pi/Main/

# The command now uses the full path as well
ExecStart=python3 /home/pi/Main/Main.py & > /home/pi/Logs/deskpallog 2>&1

# Restart the service if it crashes
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target
"


sudo systemctl daemon-reload

sudo systemctl enable deskpal.service

sudo systemctl start deskpal.service


*** at the bottom of the file, put the following line: ***
@reboot sh /home/pi/Launch/Launcher.sh >/home/pi/Logs/deskpallog 2>&1

*****deskpal.service will log any errors or stasuses into /home/pi/Logs/deskpallog*****

***Test the process by rebooting***
sudo reboot
